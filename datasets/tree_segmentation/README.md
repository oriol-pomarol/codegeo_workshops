# 🌳 Tree Crown Segmentation Dataset

This folder contains data for tree crown segmentation, derived from the [Quebec Trees Dataset](https://doi.org/10.5281/zenodo.8148479).  
The version provided here corresponds to the one used in the tree crown segmentation contest described in the following paper:

- [ISPRS Archives, 2024: "Benchmarking Tree Crown Segmentation in Airborne Images"](https://doi.org/10.5194/isprs-archives-XLVIII-3-2024-637-2024)  
  *(Note: This paper describes the contest and dataset modifications, but is not itself a dataset.)*
